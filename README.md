# Oslo Hi-fi Center API
An all purpose API for Oslo Hi-fi Center administration with the following directory structure

```bash
├── db.sqlite3
├── manage.py
├── ohc
│   ├── __init__.py
│   ├── __pycache__
│   ├── asgi.py
│   ├── settings.py
│   ├── urls.py
│   └── wsgi.py
└── posts
    ├── __init__.py
    ├── __pycache__
    ├── admin.py
    ├── apps.py
    ├── migrations
    │   ├── 0001_initial.py
    │   ├── 0002_rename_titles_post_title_rename_poster_vote_post.py
    │   ├── __init__.py
    │   └── __pycache__
    ├── models.py
    ├── serializers.py
    ├── tests.py
    └── views.py
```
